Add to ~/.cursor/mcp.json (or Settings → MCP → Add new MCP server), then click Authorize in the browser tab that opens.
{
  "mcpServers": {
    "tuckit": { "url": "{{ mcp_url }}" }
  }
}
Run these, then approve in the browser tab that opens.
codex mcp add tuckit --url {{ mcp_url }}

Antigravity's OAuth handshake for remote MCP servers is unreliable right now, so use a header token instead — create one under “Access tokens · Headless & CI” below, then add it to mcp_config.json (Settings → Customizations → Open MCP Config):
{
  "mcpServers": {
    "tuckit": {
      "serverUrl": "{{ mcp_url }}",
      "headers": { "Authorization": "Bearer YOUR_TUCKIT_TOKEN" }
    }
  }
}
